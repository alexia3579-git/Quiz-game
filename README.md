A simple and interactive command-line quiz game written in Python!
This project quizzes the user on basic computer hardware terminology and tracks their score based on correct answers. It's a great beginner-friendly example of reading user input, using conditionals, and working with basic logic in Python.

Features:

-Interactive question-and-answer gameplay
-Case-insensitive answer checking
-Score tracking
-Percentage calculation at the end
-Very beginner-friendly

How It Works:

The game asks the user four questions:

1. What does CPU stand for? *Central Processing Unit
2. What does GPU stand for? *Graphics Processing Unit
3. What does RAM stand for? *Random Access Memory
4. What does PSU stand for? *Power Supply Unit

For each correct response, the user earns 1 point.

At the end, the program displays:
-Total number of correct answers
-Final score percentage
